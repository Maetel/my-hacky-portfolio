# My Hacky Portfolio

* 100% native canvas API used
* Built with typescript & esbuild

> Up and running [here (https://my-hacky-portfolio.vercel.app/)](https://my-hacky-portfolio.vercel.app/)

```shell
# install
bun i

# dev - change script src in public/index.html
bun dev

# build
bun run build
```
